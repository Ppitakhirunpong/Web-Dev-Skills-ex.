WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language is the content of web page.
Example of HTML include heading, paragraph, images, link, lists, etc.

Check out the HTML tutorial at w3schools.com.

Learn CSS

CSS stands for Cascading Style Sheets CSS describes how HTML elements are
to be displayed on screen, paper, or in other media CSS saves a lot of work. It can
control the layout of multiple web pages all at once External stylesheets are stored in CSS files

Check out the CSS tutorial at w3schools.com.

Learn JavaScript

JavaScript is the programming language of the web . JavaScript helps to make
web pages interactive and dynamic. Some common Javascript taskd are:

  • Add new HTML tp the page,
  • change the existing content / style,
  • react to user events (mouse clicks, key presses, etc.)

Check out the JavaScript tutorial at w3schools.com.
